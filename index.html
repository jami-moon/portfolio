Skills

PUBLISHING SKILL
HTML

의미에 맞는 태그를 활용하는 시멘틱한 마크업에 관심이 많습니다. BEM방법론 등을 활용하여 유지보수를 고려한 id 및 class 작성이 가능합니다. 웹 접근성을 위한 WAI-ARIA 속성 등을 공부하고 있습니다.

CSS

Class 제어를 통해 여러가지 시각적인 효과를 표현할 수 있습니다. style-lint 등의 linter를 활용해 올바르고, 순서있는 CSS를 작성할 수 있습니다. 일반적인 반응형 작업 외에도 그리드 시스템을 활용한 반응형 작업을 한 경험이 있습니다.

SCSS

CSS 모듈화에 관심이 생겨 SCSS를 배웠으며 SCSS를 활용한 코드 재사용 및 함수, 연산자 활용이 가능합니다. 최근 업데이트된 dart-sass를 통해 CSS파일로의 컴파일이 가능합니다.

JavaScript

자바스크립트,제이쿼리의 DOM 제어를 통해 여러가지 시각적인 효과를 표현할 수 있습니다. 네비게이션과 메뉴 등에서 사용되는 기능들을 순수 자바스크립트, 제이쿼리로 작성할 수 있습니다. 여러가지 자바스크립트, 제이쿼리 플러그인을 공식문서를 참조하여 활용할 수 있습니다.
